# live_exercise

## Setup
1. Open google colab: [Google Colab Link](https://colab.research.google.com/)
2. Open the notebook: [live_exercise.ipynb](https://colab.research.google.com/github/jordi-marimon/live_exercise/blob/main/live_exercise.ipynb) in google collab.
    ctrl+O > Github
    Enter the repository URL: `https://github.com/jordi-marimon/live_exercise.git`
    Select the notebook file: `live_exercise.ipynb` from the UI.

3. The notebook is ready to run in Google Colab, the first cells will take care of installing the required dependencies.

## Tests
In order to check the correctness of your implementation, you can run the test cells at the bottom of the notebook. They will validate your code against several test cases to check whether your implementation is correct or not.
